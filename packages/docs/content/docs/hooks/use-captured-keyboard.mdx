---
title: useCapturedKeyboard
description: Capture keyboard events for a selected focusable element
---

The `useCapturedKeyboard` hook captures keyboard events for a selected focusable element
and exposes a callback to forward them to child components (e.g. `PromptInput`).

It combines `useKeyboard` with `selectedOnly: true` and a ref-based capture pattern
into a single hook call.

```tsx title="Import"
import { useCapturedKeyboard } from "@gridland/utils"
```

## Usage

```tsx title="Forward keyboard to PromptInput"
function FocusablePrompt({ onSubmit, status }: Props) {
  const { isFocused, isSelected, focusId, focusRef } = useFocus({ id: "prompt" })
  const captureKeyboard = useCapturedKeyboard(focusId)

  return (
    <box ref={focusRef}>
      <PromptInput
        onSubmit={onSubmit}
        status={status}
        focus={isSelected}
        useKeyboard={captureKeyboard}
      />
    </box>
  )
}
```

The returned `capture` function is passed as the `useKeyboard` prop to a child component.
The child registers its key handler via the capture function, and `useCapturedKeyboard`
forwards keyboard events to it only when the parent element is selected.

## Examples

### With TextInput

Forward keyboard to a `TextInput` inside a focusable panel.

```tsx title="Captured TextInput"
function EditablePanel() {
  const { isFocused, isSelected, focusId, focusRef } = useFocus({ id: "editor" })
  const captureKeyboard = useCapturedKeyboard(focusId)
  const [value, setValue] = useState("")

  return (
    <box ref={focusRef} border>
      <TextInput
        label="Name"
        value={value}
        onChange={setValue}
        focus={isSelected}
        useKeyboard={captureKeyboard}
      />
    </box>
  )
}
```

### With SelectInput

Use the same pattern for any interactive child.

```tsx title="Captured SelectInput"
function ModelPicker() {
  const { isSelected, focusId, focusRef } = useFocus({ id: "model" })
  const captureKeyboard = useCapturedKeyboard(focusId)

  return (
    <box ref={focusRef} border>
      <SelectInput
        items={[
          { label: "Sonnet", value: "sonnet" },
          { label: "Opus", value: "opus" },
        ]}
        focus={isSelected}
        useKeyboard={captureKeyboard}
        onSubmit={(v) => console.log("Selected:", v)}
      />
    </box>
  )
}
```
