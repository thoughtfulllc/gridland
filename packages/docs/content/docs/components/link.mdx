The `Link` component from `@polyterm.io/ui` renders a clickable hyperlink in the
terminal UI.

<div className="my-8">
  <LinkDemo />
</div>

## Run demo

```bash title="Terminal"
curl -fsSL https://raw.githubusercontent.com/cjroth/polyterm/main/scripts/run-demo.sh | bash -s link
```

## Installation

<Tabs items={['bun', 'npm', 'yarn', 'pnpm']}>

## Usage

```tsx title="Example"
import { Link } from "@polyterm.io/ui"

<Link url="https://opentui.com">Visit opentui.com</Link>
```

## Props

| Prop | Type | Description |
|------|------|-------------|
| `children` | `ReactNode` | Link text content |
| `url` | `string` | Target URL |
| `underline` | `"solid" \| "dashed" \| "dotted" \| "none"` | Underline style (default `"solid"`) |
