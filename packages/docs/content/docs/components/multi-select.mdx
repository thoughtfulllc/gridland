The `MultiSelect` component from `@polyterm.io/ui` renders a multi-selection list
with checkbox indicators, keyboard navigation, and controlled/uncontrolled modes.

## Usage

```tsx title="Example"
import { MultiSelect } from "@polyterm.io/ui"
import { useKeyboard } from "@opentui/react"

<MultiSelect
  items={[
    { label: "TypeScript", value: "ts" },
    { label: "JavaScript", value: "js" },
    { label: "Python", value: "py" },
  ]}
  useKeyboard={useKeyboard}
  onSubmit={(selected) => console.log("Selected:", selected)}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `MultiSelectItem<V>[]` | `[]` | Array of selectable items |
| `selected` | `MultiSelectItem<V>[]` | — | Controlled selected items |
| `defaultSelected` | `MultiSelectItem<V>[]` | `[]` | Initially selected items |
| `focus` | `boolean` | `true` | Whether the list is focused |
| `initialIndex` | `number` | `0` | Initially highlighted index |
| `limit` | `number` | — | Max visible items (enables scrolling) |
| `onSelect` | `(item) => void` | — | Called when an item is checked |
| `onUnselect` | `(item) => void` | — | Called when an item is unchecked |
| `onSubmit` | `(items) => void` | — | Called on Enter key with all selected items |
| `onHighlight` | `(item) => void` | — | Called when highlighted item changes |
| `useKeyboard` | `function` | — | Keyboard hook from `@opentui/react` |

## Controls

- **Up/Down** or **j/k**: Navigate items
- **Space**: Toggle selection
- **Enter**: Submit selected items
