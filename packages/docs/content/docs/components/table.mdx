The `Table` component from `@polyterm.io/ui` renders structured data in a bordered table
with automatic column detection and box-drawing characters.

## Usage

```tsx title="Basic usage"
import { Table } from "@polyterm.io/ui"

<Table
  data={[
    { name: "Alice", age: 30, role: "Engineer" },
    { name: "Bob", age: 25, role: "Designer" },
  ]}
/>
```

```tsx title="Custom columns and colors"
<Table
  data={[
    { name: "Alice", age: 30, role: "Engineer" },
    { name: "Bob", age: 25, role: "Designer" },
  ]}
  columns={["name", "role"]}
  headerColor="cyan"
  borderColor="#5e81ac"
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `data` | `T[]` | — | Array of data objects to display |
| `columns` | `(keyof T)[]` | — | Keys to display as columns (auto-detected from data if omitted) |
| `padding` | `number` | `1` | Cell padding in characters |
| `headerColor` | `string` | `"blue"` | Foreground color for header text |
| `borderColor` | `string` | — | Color for border characters |
