The `TextInput` component from `@polyterm.io/ui` provides an interactive text input
with customizable prompt, placeholder, and controlled/uncontrolled modes.

## Usage

```tsx title="Uncontrolled"
import { TextInput } from "@polyterm.io/ui"

<TextInput
  placeholder="Type something..."
  onSubmit={(value) => console.log("Submitted:", value)}
/>
```

```tsx title="Controlled"
const [value, setValue] = useState("")

<TextInput
  value={value}
  onChange={setValue}
  onSubmit={(v) => console.log("Submitted:", v)}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | — | Controlled value |
| `onChange` | `(value: string) => void` | — | Called when value changes |
| `onSubmit` | `(value: string) => void` | — | Called on Enter key |
| `placeholder` | `string` | `""` | Placeholder text |
| `prompt` | `string` | `"> "` | Prompt string displayed before input |
| `promptColor` | `string` | `"cyan"` | Color of the prompt text |
| `focus` | `boolean` | `true` | Whether the input is focused |
| `maxLength` | `number` | — | Maximum input length |
