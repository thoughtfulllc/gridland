## Usage

```tsx title="Basic"
import { ChatInput } from "@gridland/ui"
import { useKeyboard } from "@opentui/react"

<ChatInput
  placeholder="Message Claude..."
  useKeyboard={useKeyboard}
  onSubmit={(text) => console.log("Sent:", text)}
/>
```

```tsx title="With slash commands"
<ChatInput
  commands={[
    { cmd: "/help", desc: "Show commands" },
    { cmd: "/model", desc: "Switch model" },
    { cmd: "/clear", desc: "Clear conversation" },
  ]}
  useKeyboard={useKeyboard}
  onSubmit={(text) => console.log(text)}
/>
```

```tsx title="With file mentions"
<ChatInput
  files={["src/index.ts", "src/routes.ts", "package.json"]}
  useKeyboard={useKeyboard}
  onSubmit={(text) => console.log(text)}
/>
```

```tsx title="Disabled while streaming"
<ChatInput
  disabled
  disabledText="Generating..."
  useKeyboard={useKeyboard}
/>
```

```tsx title="Custom suggestion provider"
<ChatInput
  getSuggestions={(value) => {
    if (value.startsWith("#")) {
      return [
        { text: "#bug", desc: "Bug report" },
        { text: "#feature", desc: "Feature request" },
      ].filter((s) => s.text.startsWith(value))
    }
    return []
  }}
  useKeyboard={useKeyboard}
  onSubmit={(text) => console.log(text)}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `onSubmit` | `(text: string) => void` | — | Called when user submits a message |
| `onChange` | `(text: string) => void` | — | Called when input value changes |
| `placeholder` | `string` | `"Type a message..."` | Placeholder text when empty |
| `prompt` | `string` | `"❯ "` | Prompt character before input |
| `promptColor` | `string` | `theme.muted` | Color of the prompt character |
| `disabled` | `boolean` | `false` | Disable input |
| `disabledText` | `string` | `"Generating..."` | Text shown when disabled |
| `commands` | `{ cmd: string; desc?: string }[]` | `[]` | Slash commands for autocomplete |
| `files` | `string[]` | `[]` | File paths for @ mention autocomplete |
| `getSuggestions` | `(value: string) => Suggestion[]` | — | Custom suggestion provider |
| `maxSuggestions` | `number` | `5` | Max visible suggestions |
| `enableHistory` | `boolean` | `true` | Enable command history |
| `showDividers` | `boolean` | `false` | Show horizontal dividers above and below input |
| `width` | `number` | `80` | Width for dividers |
| `useKeyboard` | `function` | — | Keyboard hook from `@opentui/react` |

### Suggestion

| Field | Type | Description |
|-------|------|-------------|
| `text` | `string` | Suggestion text |
| `desc` | `string?` | Optional description |

## Controls

- **Enter**: Submit message (or accept suggestion)
- **Tab**: Cycle through suggestions
- **↑/↓**: Navigate suggestions or command history
- **Escape**: Dismiss suggestions
- **/** : Trigger slash command suggestions
- **@**: Trigger file mention suggestions
