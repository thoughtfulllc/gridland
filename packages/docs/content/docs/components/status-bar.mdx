The `StatusBar` component from `@polyterm.io/ui` renders a horizontal bar that
displays keybinding hints showing key combinations and their associated actions.


## Usage

```tsx title="Example"
import { StatusBar } from "@polyterm.io/ui"

<StatusBar
  items={[
    { key: "Tab", label: "switch focus" },
    { key: "q", label: "quit" },
  ]}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `StatusBarItem[]` | -- | Array of keybinding hints to display |
| `extra` | `ReactNode` | `undefined` | Optional inline content rendered left of the hints (must be `<span>` or string) |

### StatusBarItem

| Field | Type | Description |
|-------|------|-------------|
| `key` | `string` | Key or key combination text |
| `label` | `string` | Description of what the key does |
