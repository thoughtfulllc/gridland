The `Spinner` component from `@polyterm.io/ui` renders an animated loading indicator
with customizable text and color.

## Installation

<Tabs items={['npm', 'yarn', 'pnpm', 'bun']}>
<Tab value="npm">
```bash title="Terminal"
npm install @polyterm.io/ui
```
</Tab>
<Tab value="yarn">
```bash title="Terminal"

```
</Tab>
<Tab value="pnpm">
```bash title="Terminal"
pnpm add @polyterm.io/ui
```
</Tab>
<Tab value="bun">
```bash title="Terminal"
bun add @polyterm.io/ui
```
</Tab>

## Usage

```tsx title="Example"
import { Spinner } from "@polyterm.io/ui"

<Spinner text="Loading..." color="cyan" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `text` | `string` | `"Loading"` | Text displayed next to the spinner |
| `color` | `string` | `"gray"` | Color of the spinner and text |
| `interval` | `number` | `100` | Animation frame interval in milliseconds |
