The `Spinner` component from `@gridland/ui` renders an animated loading indicator
with 5 built-in animation styles selectable via the `variant` prop.

<div className="my-8">
  <SpinnerPickerDemo />
</div>

### Available variants

`dots` · `pulse` · `meter` · `bloom` · `ellipsis`

  <SpinnerShowcaseDemo />
</div>

## Installation

```bash
bunx shadcn@latest add https://raw.githubusercontent.com/cjroth/gridland/main/packages/ui/registry.json spinner
```

## Usage

```tsx title="Basic"
import { Spinner } from "@/components/ui/spinner"

<Spinner text="Loading..." />
```

<div className="my-4">
  <SpinnerBasicDemo />
</div>

```tsx title="With variant"
<Spinner variant="bloom" text="Fetching data..." />
<Spinner variant="pulse" text="Building" color="#05FFA1" />
<Spinner variant="ellipsis" text="Please wait" />
```

## Props

### Spinner

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `SpinnerVariant` | `"dots"` | Animation style |
| `text` | `string` | - | Text displayed next to the spinner |
| `color` | `string` | `theme.accent` | Foreground color of the spinner frame |

### SpinnerPicker

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `useKeyboard` | `(handler) => void` | - | Keyboard handler from `@opentui/react` for ←→ navigation |
