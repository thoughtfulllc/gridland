The `Modal` component from `@polyterm.io/ui` renders a bordered container that overlays
content, optionally displays a title, and listens for the Escape key to trigger a close callback.

<div className="my-8">

</div>

## Run demo

```bash title="Terminal"
curl -fsSL https://raw.githubusercontent.com/cjroth/polyterm/main/scripts/run-demo.sh | bash -s modal
```

## Usage

```tsx title="Example"
import { Modal } from "@polyterm.io/ui"

<Modal title="Settings" borderColor="blue" onClose={() => setOpen(false)}>
  <text>Modal content here</text>
</Modal>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | -- | Content rendered inside the modal |
| `title` | `string` | `undefined` | Optional title at the top of the border |
| `borderColor` | `string` | `"blue"` | Color of the border and title text |
| `borderStyle` | `"single" \| "double" \| "rounded" \| "heavy"` | `"rounded"` | Border character set |
| `onClose` | `() => void` | `undefined` | Callback invoked on Escape key press |
| `useKeyboard` | `(handler) => void` | `undefined` | Keyboard handler hook |
