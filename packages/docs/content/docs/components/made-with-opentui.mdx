The `CornerRibbon`, `BadgeButton`, and `TextBadge` components from `@polyterm.io/ui` provide
branding attribution that links back to the OpenTUI project website.

## Import

```tsx title="Import"
import { CornerRibbon, BadgeButton, TextBadge } from "@polyterm.io/ui"
```

## Usage

```tsx title="CornerRibbon"
<CornerRibbon position="top-right" />
```

```tsx title="BadgeButton"
<BadgeButton variant="dark" />
```

```tsx title="TextBadge"
<TextBadge />
```

## CornerRibbon Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `position` | `"top-right" \| "top-left" \| "bottom-right" \| "bottom-left"` | `"top-right"` | Corner position |
| `className` | `string` | `undefined` | Additional CSS classes on the ribbon link |
| `absolute` | `boolean` | `false` | Use absolute instead of fixed positioning |

## BadgeButton Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"dark" \| "light" \| "outline"` | `"dark"` | Visual style variant |
| `className` | `string` | `undefined` | Additional CSS classes |

## TextBadge Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | `undefined` | Additional CSS classes |
