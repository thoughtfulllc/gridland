The `SelectInput` component from `@polyterm.io/ui` renders a keyboard-navigable
selection list with arrow key support.

## Import

```tsx title="Import"
import { SelectInput } from "@polyterm.io/ui"
```

## Usage

```tsx title="Example"
<SelectInput
  items={[
    { label: "Option A", value: "a" },
    { label: "Option B", value: "b" },
    { label: "Option C", value: "c" },
  ]}
  onSelect={(item) => console.log("Selected:", item.value)}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `SelectInputItem<V>[]` | `[]` | Array of selectable items |
| `focus` | `boolean` | `true` | Whether the list is focused for keyboard input |
| `initialIndex` | `number` | `0` | Initially highlighted index |
| `limit` | `number` | — | Max visible items (enables scrolling) |
| `onSelect` | `(item: SelectInputItem<V>) => void` | — | Called when an item is selected (Enter) |
| `onHighlight` | `(item: SelectInputItem<V>) => void` | — | Called when highlighted item changes |
| `textColor` | `string` | — | Default text color |
| `selectedTextColor` | `string` | — | Text color for selected item |
| `focusedTextColor` | `string` | — | Text color for focused item |
| `backgroundColor` | `string` | — | Default background color |
| `selectedBackgroundColor` | `string` | — | Background color for selected item |
| `focusedBackgroundColor` | `string` | — | Background color for focused item |

## Item Shape

```tsx
type SelectInputItem<V> = {
  key?: string
  label: string
  value: V
}
```
