The `Ascii` component from `@polyterm.io/ui` renders text as ASCII art using
various font styles.

<div className="my-8">
  <AsciiDemo />
</div>

## Run demo

```bash title="Terminal"
curl -fsSL https://raw.githubusercontent.com/cjroth/polyterm/main/scripts/run-demo.sh | bash -s ascii
```

## Usage

```tsx title="Example"
import { Ascii } from "@polyterm.io/ui"

<Ascii text="Hello" font="slick" color="#88c0d0" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `text` | `string` | — | Text to render as ASCII art |
| `font` | `"tiny" \| "block" \| "slick" \| "shade"` | — | ASCII font style |
| `color` | `string` | — | Foreground color |
