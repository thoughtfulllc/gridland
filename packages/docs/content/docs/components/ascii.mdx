The `Ascii` component from `opentui-ui` renders text as ASCII art using
various font styles.

## Import

```tsx title="Import"
import { Ascii } from "opentui-ui"
```

## Usage

```tsx title="Example"
<Ascii text="Hello" font="slick" color="#88c0d0" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `text` | `string` | — | Text to render as ASCII art |
| `font` | `"tiny" \| "block" \| "slick" \| "shade"` | — | ASCII font style |
| `color` | `string` | — | Foreground color |

## Font Examples

